About

Welcome to the Know.edu Admin Panel! Here, you will be able to make changes to the website, and view analytics and statistics. To get started, click on any of the options on the sidebar to the left. You are currently able to view teachers, delete/ban users for misconduct, Search enrollments, edit courses and view analytics.

Clicking the above image will guide you to the main website. Remember to be very careful when using the admin panel, as changes made will affect the whole website, and irresponsible or accidental usage could result in serious data loss. In case of such a scenario, contact an administrator of higher authority, who has access to the restore data function. Click the button below to view database.
